疑難排解

手電筒無法使用？

  1. 確認已允許相機權限（瀏覽器會彈出提示）
  2. 確認沒有其他應用程式正在使用相機
  3. 嘗試重新整理頁面
  4. 使用 Chrome 或 Edge 瀏覽器（Android）
  5. 確認使用 HTTPS 連線或本地伺服器

位置無法獲取？

  1. 確認已允許位置權限
  2. 確認手機 GPS 已開啟
  3. 嘗試在戶外或窗邊使用
此應用程式需要位置和相機權限才能正常運作。所有資料僅在您的裝置上處理，不會上傳到伺服器。
